# Contributing to Quantum Kienzy

Thank you for your interest in contributing to Quantum Kienzy!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/quantum-kienzy.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Commit: `git commit -m "feat: add your feature"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

```bash
pnpm install
pnpm run dev
```

## Code Style

- Use TypeScript
- Follow ESLint rules
- Format with Prettier: `pnpm run format`

## Testing

```bash
pnpm run test
```

## Commit Messages

- `feat:` for new features
- `fix:` for bug fixes
- `docs:` for documentation
- `style:` for formatting
- `refactor:` for code refactoring
- `test:` for tests

## Pull Request Process

1. Update README if needed
2. Add tests for new features
3. Ensure all tests pass
4. Request review from maintainers
